# LittleWitch

Game project

A 3rd-person action platformer. Player uses an array of magical spells to solve puzzles, clear platforming challenges, and defeat enemies.


Features implemented:
- Fireball spell
- Timeslow spell
- Freeze spell
- Levitation spell
- Animations
- 1 Enemy AI type
- Moving platforms


Made with Unity

Status: Abandoned (due to scope size)

Will not be updated
